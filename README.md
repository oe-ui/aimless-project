# Aimless Project

It's a simple project to test ideas.

## Tech Stack

- Build Tools: `Vite`
- Language: `TypeScript`
- F/E Framework: `Astro`
- Lint & Format: `Biome`
- Testing: `Vitest`

## Project Structure

- `apps/`: Applications (web, mobile, desktop etc.)
- `docs/`: Documentation (storybook, docusaurus, etc.)
- `packages/`: Shared packages (components, libraries, entities etc.)
